pdfdiff — How to use

1

Load your two PDFs

Drag & drop or click the upload zones. A = old version, B = new version. Any PDF works: technical drawings, text documents, mixed content. Multi-page PDFs are supported — use the ◀ ▶ buttons or ← → keys to flip pages.

2

Navigation — zoom and pan

Mouse wheel = zoom (focused on the cursor). Drag (left click) = pan. On mobile: pinch = zoom, one-finger drag = pan. PDFs render at vector quality — zoom in as far as you like, text and lines stay razor sharp. A "Rendering..." badge shows while the sharp redraw runs.

3

Comparison — 3 modes

Overlay

Both PDFs drawn on top of each other. The opacity slider controls how visible B is. The Blink button flashes B on and off — the fastest way to spot small differences (text changes, dimension tweaks).

Side by side

A and B next to each other at the same zoom level. Natural visual comparison.

Swipe

A draggable vertical divider: A on the left, B on the right. Ideal for drawings — differences pop instantly along the line.

4

Manual alignment

If the two PDFs were saved in different coordinate systems (different origin, paper size or plot offset): use ⊞ Align to drag the B layer onto A. Drag the orange dashed B layer with the mouse, fine-tune with arrow keys (1px), Shift+arrow = 10px. For pixel-perfect alignment use ⌖ Point align: zoom in, click a distinctive point (corner, intersection) on A, then the same point on B — the tool snaps them together exactly. For rotated or skewed scans, use the rotate buttons (⟲/⟳ 90° for landscape↔portrait, ⟲/⟳ 1° for fine-tuning, Shift+click = 0.1°) or ⌖⌖ 2-Point, which computes rotation AND translation from two point pairs. "All pages" applies the alignment to every page. Enter or ✓ Done exits.

5

Text diff — automatic label comparison (key feature)

The Text diff button reads every text label on both drawings (dimensions, elevations, annotations, descriptions), matches them up, and lists what changed, what's new, what was removed — e.g. "25.0 × 5.0 m → 25.0 × 6.0 m". Matching uses a local alignment field, so it works even when the two drawings sit in shifted coordinate systems. Every difference gets a confidence %, low-confidence noise can be hidden, and clicking an item zooms right to it on the drawing. Requires a real text layer in the PDF (CAD exports usually have one).

6

AI summary (key feature)

Inside the text diff panel, the AI button turns the detected changes into a professional summary: it names the affected element (e.g. "the stormwater reservoir grew"), categorizes each change, rates its significance and tells you what to double-check. Works with Claude (Anthropic) or ChatGPT (OpenAI) — bring your own API key. Privacy: by default only the text differences are sent (never the drawing itself), and you see the exact payload before sending. The deterministic engine finds the changes — the AI only interprets them.

7

Measure, annotate, export

Measure: calibrate once against a known length (e.g. a dimension line), then measure any real-world distance on the drawing. Drawing tools (line, rectangle, circle, arrow, text) let you mark up differences by hand. ↩ = undo, ✕ = clear. Export saves the current view as PNG with your annotations. Report (in the text diff panel) generates a downloadable PDF of all changes — with the AI summary if you ran one.

8

Pro tips

• Text diff is the fastest way to get an exact list of dimension and label changes — it handles shifted coordinates too.
• Blink is best for visual/geometric differences — the human eye is excellent at catching flicker.
• Suggested workflow: Text diff for the labels → Blink for the geometry → AI for the summary → Report for the meeting.
• Use ◀ ▶ in the text diff panel to step through every change like a checklist.
• Hold Space to pan while a drawing tool is active.

•Your PDFs never leave your device. All processing happens in your browser. Nothing is uploaded to any server.
